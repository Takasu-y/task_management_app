# task-manage-app
カンバン方式のタスク管理アプリです。
trelloのようにセクション毎でタスクを管理することが出来ます。
また、ドラッグアンドドロップでタスクの並べ替えやセクション間の移動が可能です。

# Usage
## section
* セクションの右側＋ボタンで新たにセクションを追加。
* セクション名はダブルクリックすることで編集出来ます。
* 各セクション下部の＋ボタンで新たにタスクを追加。

## task
タスクカードはタスクのタイトルと内容を記入して下さい。
* 編集：🖋アイコンまたはタスクカードをダブルクリック
* お気に入り：★アイコンをクリック
* チェック：✔️アイコンをクリック
* 削除：🗑アイコンをクリック
